## Config
You can configure the project using .env file. Following config options are available.
```

SNIPCART_KEY=
GATSBY_PORTAL_ID=
GATSBY_FORM_ID=
GATSBY_COFFEE_USERNAME=
GATSBY_GITHUB_REPO_LINK=
```
